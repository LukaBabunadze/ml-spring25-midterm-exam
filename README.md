## ­ЪЊѓ ML- рЃерЃБрЃљрЃџрЃћрЃЊрЃБрЃарЃў рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃљ/task_01

### ­ЪЊё pearson.py

рЃћрЃА рЃАрЃЎрЃарЃўрЃърЃбрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА NumPy рЃЊрЃљ Matplotlib рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃА, рЃарЃљрЃЌрЃљ рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃЮрЃА рЃЊрЃљ рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ рЃњрЃљрЃБрЃЎрЃћрЃЌрЃЮрЃА **рЃърЃўрЃарЃАрЃЮрЃюрЃўрЃА рЃЎрЃЮрЃарЃћрЃџрЃљрЃфрЃўрЃўрЃА рЃЎрЃЮрЃћрЃцрЃўрЃфрЃўрЃћрЃюрЃбрЃА** рЃЮрЃа рЃфрЃЋрЃџрЃљрЃЊрЃА рЃерЃЮрЃарЃўрЃА. рЃљрЃАрЃћрЃЋрЃћ, рЃљрЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃА рЃарЃћрЃњрЃарЃћрЃАрЃўрЃљрЃА рЃЊрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃЊрЃўрЃљрЃњрЃарЃљрЃЏрЃљрЃА.

### ­ЪћД рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў

- рЃўрЃЌрЃЋрЃџрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃА
- рЃљрЃдрЃарЃўрЃфрЃ«рЃљрЃЋрЃА рЃњрЃљрЃЊрЃљрЃ«рЃЋрЃћрЃЋрЃћрЃЉрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃЮрЃЊрЃљрЃю
- рЃљрЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃА рЃЎрЃЮрЃЋрЃљрЃарЃўрЃљрЃфрЃўрЃўрЃА рЃЊрЃљ рЃЊрЃўрЃАрЃърЃћрЃарЃАрЃўрЃўрЃА рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃА
- рЃўрЃЌрЃЋрЃџрЃўрЃА **рЃърЃўрЃарЃАрЃЮрЃюрЃўрЃА рЃЎрЃЮрЃарЃћрЃџрЃљрЃфрЃўрЃўрЃА рЃЎрЃЮрЃћрЃцрЃўрЃфрЃўрЃћрЃюрЃбрЃА (r)**
- рЃљрЃерЃћрЃюрЃћрЃЉрЃА рЃЏрЃљрЃарЃбрЃўрЃЋ рЃарЃћрЃњрЃарЃћрЃАрЃўрЃљрЃА (рЃњрЃљрЃарЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃўрЃА рЃ«рЃљрЃќрЃА)
- рЃњрЃљрЃЏрЃЮрЃ░рЃДрЃљрЃЋрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃгрЃћрЃарЃбрЃўрЃџрЃћрЃЉрЃў рЃЊрЃљ рЃарЃћрЃњрЃарЃћрЃАрЃўрЃўрЃА рЃ«рЃљрЃќрЃў рЃњрЃарЃљрЃцрЃўрЃЎрЃќрЃћ
- рЃљрЃџрЃљрЃЏрЃљрЃќрЃћрЃЉрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљрЃА (grid, axis repositioning, legend)

### ­ЪЊі рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЋрЃљрЃџрЃў

рЃњрЃарЃљрЃцрЃўрЃЎрЃў, рЃАрЃљрЃЊрЃљрЃф рЃюрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃўрЃљ:
- рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃгрЃћрЃарЃбрЃўрЃџрЃћрЃЉрЃў
- рЃарЃћрЃњрЃарЃћрЃАрЃўрЃўрЃА рЃ«рЃљрЃќрЃў
- рЃЎрЃЮрЃарЃћрЃџрЃљрЃфрЃўрЃўрЃА рЃЎрЃЮрЃћрЃцрЃўрЃфрЃўрЃћрЃюрЃбрЃў `r`
